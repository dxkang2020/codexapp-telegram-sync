# CodexApp Telegram Sync

[English](README.en.md) | [简体中文](README.zh-CN.md)

Keep your current Codex desktop conversation in sync with Telegram.

Use this when:

- Codex desktop is your main workspace
- Telegram is your mobile side channel when you're away from your computer
- You want Telegram to mirror the current conversation instead of managing threads

What this package includes:

- `SKILL.md` for Codex agents
- `scripts/` for the actual bridge

Current support:

- macOS: verified
- Windows: same setup style, but not yet verified on a real machine
